Lambda và Stream trong Java 8: Hướng dẫn nhanh

Giới thiệu ngắn gọn về lambda và Stream API trong Java 8, kèm ví dụ thực tế.

Dec 25, 2025
1 minute read

Trong Java 8, lambda và Stream API giúp viết mã ngắn gọn, dễ đọc và theo phong cách lập trình hàm.

  • Lambda: Biểu thức hàm gọn nhẹ thay cho class ẩn danh.
  • Stream: Xử lý tập dữ liệu theo pipeline (map, filter, reduce).

Ví dụ đơn giản với lambda:

List<String> names = List.of("An", "Bình", "Chi");
names.forEach(n -> System.out.println(n));

Ví dụ Stream lọc và chuyển đổi:

List<Integer> numbers = List.of(1,2,3,4,5);
List<Integer> evenSquares = numbers.stream()
    .filter(n -> n % 2 == 0)
    .map(n -> n * n)
    .toList();

Một pipeline phổ biến:

  1. Nguồn dữ liệu: collection.stream()
  2. Trung gian: filter, map, sorted
  3. Tổng hợp: collect, reduce, count

Lưu ý hiệu năng: stream() là lazy, chỉ thực thi khi gặp operation tổng hợp. Dùng parallelStream() cho danh sách lớn nhưng cần đo đạc trước.

Khi nào dùng:

  • Duyệt danh sách + chuyển đổi dữ liệu.
  • Thay thế vòng lặp lồng nhau bằng pipeline rõ ràng.

Kết luận: Lambda + Stream giúp code Java hiện đại, súc tích và an toàn hơn.
