Async/Await trong JavaScript: Viết code bất đồng bộ dễ hiểu

Giải thích async/await và cách chuyển từ Promise để code gọn hơn.

Dec 25, 2025
1 minute read

async/await giúp bạn viết code bất đồng bộ trông như đồng bộ, giảm callback hell.

Ví dụ Promise:

fetch('/api/users')
  .then(r => r.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

Chuyển sang async/await:

async function loadUsers() {
  try {
    const r = await fetch('/api/users');
    const data = await r.json();
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}

Mẹo:

  • Bọc logic trong try/catch để bắt lỗi.
  • Dùng Promise.all khi cần chạy song song:
const [posts, comments] = await Promise.all([
  fetch('/api/posts').then(r => r.json()),
  fetch('/api/comments').then(r => r.json()),
]);

Lưu ý: await chỉ hoạt động trong hàm async. Tránh await trong vòng lặp lớn nếu không cần; hãy gom lại bằng Promise.all để tăng tốc.
