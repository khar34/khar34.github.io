Closure trong JavaScript: Bí quyết viết hàm thông minh

Tìm hiểu closure, scope và ứng dụng trong việc tạo factory, private state.

Dec 25, 2025
1 minute read

Closure là khả năng một hàm ghi nhớ phạm vi (scope) nơi nó được tạo, dù được gọi ở nơi khác.

Ví dụ tạo counter riêng tư:

function createCounter() {
  let count = 0; // private
  return {
    inc() { count++; },
    get() { return count; }
  };
}

const c1 = createCounter();
c1.inc();
console.log(c1.get()); // 1

Ứng dụng:

  • Factory pattern tạo đối tượng có state riêng.
  • Tạo hàm cấu hình: makeAdder(5) trả về hàm cộng 5.
  • Tránh lộ biến toàn cục, giữ code sạch.

Lưu ý:

  • Tránh lạm dụng closure gây rò rỉ bộ nhớ do tham chiếu không cần thiết.
  • Hiểu rõ block scope (let, const) vs function scope (var).

Kết luận: Nắm closure giúp bạn viết JS linh hoạt, tránh bug liên quan phạm vi biến.
